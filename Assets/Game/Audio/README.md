# Audio

Contains all audio of the game. Please use a subfolder to put your asset.

## Subfolders

* Dialog: Contains all dialogs of the game per level.
* FX: Contains al the sound fx of the game.
* Music: Contains the game music per level.
* Scripts: Very specialized scripts related only to audio and reusable per game.
* Sound: Any other sound that do not clasify in the other subfolders.
